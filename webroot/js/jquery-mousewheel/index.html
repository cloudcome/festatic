jquery-mousewheel.js docs

作者：云淡然

兼容：ie9+、chrome、firefox

已适配yquery

1、插件说明
  1. 鼠标向上滚动为+1，向下滚动为-1。
  2. 采用定时器监听滚动的过程和结束。
2、插件API

2.1、插件参数

$.fn.mousewheel.defaults;
参数名称 参数默认值 参数描述
timeout 456

延时监听滚动停止事件的时间（单位毫秒）
isPreventDefault true

是否阻止默认事件，默认true
onmousewheelstart $.noop

开始滚动回调

this：element
onmousewheel $.noop

正在滚动回调

this：element

参数1：滚动的方向，1：上，-1：下
onmousewheelend $.noop

滚动停止回调

this：element

参数1：滚动的距离

2.2、插件方法

$('#demo').mousewheel('name', 'args0', 'args1', ...);
方法名称 方法参数 方法描述
preventDefault

参数1：isPreventDefault，布尔值

是否阻止默认事件。
