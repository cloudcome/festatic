作者：云淡然

兼容：ie10（含）+、chrome、firefox

1、预热

1.1、特点

  1. 相关文章：暂无。
  2. 兼容ie10（含）+、chrome、firefox。
  3. 包含了移动端的事件。
  4. 最精简、最常用的API，大部分API和jQuery的使用方法一致。

1.2、怎么和jQuery、zepto相比

  1. 为标准而生，不知道IE9(lte)是什么，我只认识chrome。
  2. 没有复杂的兼容考虑，最精简的代码，适合现代浏览器。与jQuery和zepto相比，适用场景略窄。
  3. 接近完美的CSS3动画。
  4. 可以直接不带浏览器前缀获取、设置CSS。
  5. 最大程度的兼容jQuery，0学习成本。
  6. 更小的代码体积，加载更快。
