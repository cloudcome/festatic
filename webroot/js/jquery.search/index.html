2、插件API

2.1、插件参数

$.search(settings);
$.search.defaults
参数名称 参数默认值 参数描述
isStrict true

是否严格模式，默认true

严格模式下，将在读取后、写入前分别进行encodeURIComponent、decodeURIComponent操作
search ""

传入search部分，为空默认为window.location.search

2.2、插件方法

$.search(settings).fn();
方法名称 方法参数 方法描述
set
  1. key, val
  2. keyValMap
 设置1个或多个search
push
  1. key, val
 向键值后插入新值
pop
  1. key
 删除数组末尾值
unshift
  1. key, val
 向键值前插入新值
shift
  1. key
 删除数组首值
concat
  1. key, array
 向数组末尾合并新数组
splice
  1. key, pos, val1, val2, ...
 剪切数组
get
  1. key
  2. keyArray
  3. undefined
 获取1个、多个或全部search
remove
  1. key
  2. keyArray
  3. undefined
 删除1个、多个或全部search
3、插件演示

3.1、参数配置

$.search.defaults = {
    // 是否严格模式，默认true
    // 严格模式下，将在读取后、写入前分别进行encodeURIComponent、decodeURIComponent操作
    isStrict: !0,
    // 传入search部分，为空默认为window.location.search
    search: ''
};
$.search(settings);

3.2、set

// 设置1个
$.search().set('a', '1');

// 设置多个
$.search().set({
    b: [1, 2, 3],
    c: 3
});

// push
$.search().push('a', 1);

// pop
$.search().pop('a');

// unshift
$.search().unshift('a', 2);

// shift
$.search().shift('a');

// concat
$.search().concat('a', [3, 4, 5]);

// splice
$.search().splice('a', 0, 1);

// splice
$.search().splice('a', 0, 0, 6, 7, 8);

3.3、get

// 获取1个
$.search().get('a');

// 获取多个
$.search().get(['a', 'b']);

// 获取全部
$.search().get();

3.4、remove

// 删除1个
$.search().remove('a');

// 删除多个
$.search().remove(['a', 'b']);

// 删除全部
$.search().remove();
