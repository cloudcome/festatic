jquery.cookie.js docs

作者：云淡然

兼容：ie6+、chrome、firefox

已适配yquery

1、插件说明
  1. 相关文章1： http://qianduanblog.com/2844.html
  2. 相关文章2： http://qianduanblog.com/1772.html
  3. 因为cookie的特殊性，只能读取key、value，其他参数不能获取到，所以在设置cookie的时候不同路径尽量使用不同的cookie名称。
  4. 插件的清空cookie功能默认清空 location.host下的cookie，如果你需要清空其他domain下的cookie需要自己指定默认参数的domain。
  5. 插件的删除cookie功能会把cookie真正删除，但某些浏览器会把这些cookie当做session cookie临时保存下来，如session一样，会在浏览器关闭的时候清除。
2、插件API

2.1、插件参数

$.cookie(settings);
参数名称 参数默认值 参数描述
isStrict true

是否以严格模式读取和设置cookie，默认true

严格模式下，将在读之后、写之前都会进行encodeURIComponent、decodeURIComponent操作
domain cookie设置域

在无域名（file协议）时候，必须设置为空才能在本地写入
expires 3600

默认cookie有效期1个小时（单位秒）
path "/"

默认cookie存储路径
secure false

是否加密cookie

2.1、插件方法

$.cookie(settings).fn();
方法名称 方法参数 方法描述
get
  1. undefined
  2. key
  3. keyArray
 获取1个cookie值、多个或全部cookie键值对
set
  1. key, val
  2. keyValMap
 设置1个cookie值、多个cookie值
remove
  1. undefined
  2. key, val
  3. keyArray
 移除1个cookie值、多个或全部cookie值
3、插件演示

3.1、获取

// 获取所有可读cookie键值对
$.cookie().get();

// 获取单个可读cookie值
$.cookie().get('a');

// 获取多个可读cookie键值对
$.cookie().get(['a', 'b']);

3.2、设置

// 设置1个可写cookie值
$.cookie().set('a', '1');

// 设置多个可写cookie值
$.cookie().set({
    a: '1',
    b: '2'
});

3.3、移除

// 删除所有可读可写cookie
$.cookie().remove();

// 删除单个可读可写cookie
$.cookie().remove('a');

// 删除多个可读可写cookie
$.cookie().remove(['a', 'b']);
