jquery-drop.js docs

作者：云淡然

兼容：ie9+、chrome、firefox

已适配yquery

1、插件特点
  1. 简单易用。
  2. jquery-drop必须依赖于jquery-drag
  3. 多种释放类型
  4. 相关文章： 《jquery插件13：jquery-drop拖拽释放排序、替换》
2、插件API

2.1、插件参数

$.fn.drop.defaults;
参数名称 参数默认值 参数描述

jquery-drag的配置参数，参考jquery-drag文档说明
type "insert"

释放类型

1、insert: 插入，默认

2、append: 追加

3、prepend: 前加

4、exchange: 交换

5、replace: 替换
containment null

目标容器对象
itemSelector ".item"

目标容器内的项目对象，用于判断指定释放位置

必须是选择器字符串，通常为 class
duration 456

动画时间，单位ms
dropRatio 0.1

可被拖拽到目标容器内，源模板被占比例

例，已有超过10%的面积被拖拽到目标容器内时，此时释放可以被置放在目标容器内
cloneClass "placeholder"

占位类

用于开始拖拽时占位
dropClass "drop"

占位类

用于释放目标占位
ondragover $.noop

拖拽到目标内回调

this: containment element

arg0: event
ondragenter $.noop

进入可被释放区域

即：与目标的交叉面积超过dropRatio比例时

type为exchange、replace时，目标为item，其他时候为containment

this: target element

arg0: event
ondragleave $.noop

离开可被释放区域

即：与目标的交叉面积超过dropRatio比例时

type为exchange、replace时，目标为item，其他时候为containment

this: target element

arg0: event
ondragout $.noop

拖拽离开目标内回调

this: containment element

arg0: event
ondrop $.noop

松开并放置目标后回调

this: containment element

arg0: event
