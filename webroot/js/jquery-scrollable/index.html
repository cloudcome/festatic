jquery-scrollable.js docs

作者：云淡然

兼容：ie9+、chrome、firefox

1、插件说明
  1. 相关文章： http://qianduanblog.com/3072.html
  2. 默认DOM结构：ul#demo>li*n
  3. 按需滚动任何列表，丰富配置，简单接口
  4. 无缝滚动不支持prev、next、onbeforechange、onafterchange
  5. 如果li元素中包含图片，那么请把插件初始化放在window.onload里
2、插件API

2.1、插件配置

参数名称 默认值 参数说明
listSelector li
  1. 列表选择器
width 'auto'
  1. 容器宽度，超过隐藏
height 'auto'
  1. 容器高度，超过隐藏
direction 'top'
  1. 滚动方向
scrollCount 1
  1. 每次滚动的项目数量
  2. 如果为1，则每次滚动1个，暂停的时候会是某一个项目的结尾
  3. 如果为0，则无缝滚动，暂停的时候可能会是某一个项目的中间
delay 2000ms
  1. 延迟时间
duration 678ms
  1. 动画时间
  2. 如果滚动数量为0，则滚动全部数量的动画时间
isAutoPlay true
  1. 是否自动播放
isHoverPause true
  1. 是否鼠标悬停暂停
onbeforechange function
  1. 变化之前回调
  2. this => element
onafterchange function
  1. 变化之后回调
  2. this => element

2.2、插件方法

方法名称 方法描述
play
  1. 播放
  2. e.g. $('#demo').scrollable('play');
pause
  1. 暂停
  2. e.g. $('#demo').scrollable('pause');
prev
  1. 上一轮
  2. e.g. $('#demo').scrollable('prev');
next
  1. 下一轮
  2. e.g. $('#demo').scrollable('next');
  1. number项目索引值
  2. duration滚动时间
  3. function完成回调
  1. 滚动至第几个
  2. e.g. $('#demo').scrollable(1,100,function(){alert(1);});
