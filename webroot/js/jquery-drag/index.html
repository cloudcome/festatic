2、插件API

2.1、插件参数

$.fn.drag.defaults;
参数名称 参数默认值 参数描述
handle null

鼠标操作区域选择器，默认为this

参数为选择器字符串
drag null

鼠标拖拽时，移动的目标，即从handle开始查找最近匹配的祖先元素

默认为this

参数为选择器字符串
axis "xy" 拖拽轴向，x：水平，y：垂直，xy：所有
cursor "move" 鼠标形状，为空时将不会自动设置
min null 拖拽对象的最小位置，格式为{left: 10, top: 10}
max null 拖拽对象的最大位置，格式为{left: 1000, top: 1000}
zIndex 9999

拖拽时的层级值
ondragbefore $.noop

拖拽开始前回调

this: drag element

arg0: event

arg1: instance
ondragstart $.noop

拖拽开始后回调

this: drag element

arg0: event

arg1: instance
ondrag $.noop

拖拽中回调

this: drag element

arg0: event

arg1: instance
ondragend $.noop

拖拽结束后回调

this: drag element

arg0: event

arg1: instance
