jquery.msg.js docs

作者：云淡然

兼容：ie9+、chrome、firefox

已适配yquery

1、插件说明
  1. 相关文章 http://qianduanblog.com/2603.html
  2. 消息没有modal效果，类似于android手机的消息，显示一段时间（3s）然后消失
  3. 在3秒期间，鼠标悬停会暂停消失，鼠标移开3秒后又会消失，不影响阅读！
2、插件API

2.1、默认参数

$.msg.defaults

2.2、插件参数

参数名称 参数默认值 参数描述
duration 123 动画时间
delay 3000

延迟消失时间，默认3s

如果有button时，则不会自动消失
buttons null

按钮集合数组

如果存在按钮，则不会自动消失

按钮数组格式为：

[{'确定': fn1, '取消': fn2}]
esc -1

指定按下 ESC 后触发的按钮，默认触发最后一个按钮事件

比如上面定义的2个按钮，定义触发取消事件，

那么此处填1，即第2个按钮索引
title "提示" 标题，为空不显示
msg "Hello World!!" 消息内容
type "muted" 消息类型:inverse/warning/error
/success/info/muted(默认)
zIndex 99999 层级
position null 消息位置，默认水平垂直居中

2.3、插件方法

使用返回的是 Msg 对象实例，该对象实例有以下方法：

方法名称 方法参数 方法说明
hide

参数0：fn 回调 可选

 关闭该消息，销毁该消息实例

2.4、使用范例

$.msg(settings);
$.msg('消息内容！');
$.msg('消息类型', '消息内容！');
$.msg().hide();
