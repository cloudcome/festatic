jquery-scrollbar.js docs

作者：云淡然

兼容：ie6+、chrome、firefox

已适配yquery

1、插件说明
  1. 在移动设备上，采用原生的滚动条
  2. 能够自动监听DOM的改变事件，一般DOM修改不需要额外render
  3. 自定义滚动条，需引入样式，默认样式为：http://festatic.aliapp.com/css/jquery-scrollbar/default.min.css
  4. 要使支持鼠标滚轮，需引用 jquery-mousewheel.js
  5. 要使支持拖动滚动条，需引用 jquery-drag.js
2、插件API

2.1、插件参数

$('#demo').scrollbar({
    参数名称: 参数值
})
参数名称 参数默认值 参数描述
width "auto" 渲染滚动条的区域宽度
height "auto" 渲染滚动条的区域高度
mousewheelAsix "y" 鼠标滚动绑定的滚动条方向，只能为“x”或“y”
duration 345 定向滚动的动画时间，单位ms
speed 10 每次滚动的距离，单位px
canCrossBoundary true

是否可以越过边界

true: 在容器内滚动条到达边界后，将把鼠标滚轮事件交给外部容器

false: 在容器内滚动鼠标事件一直紧抓不放

2.2、插件方法

$('#demo').scrollbar('方法名称', 方法参数)
方法名称 方法参数 方法描述
"x" 无

获取当前x轴滚动区域占比

值在[0,1]之间
"y" 无

获取当前y轴滚动区域占比

值在[0,1]之间
"x"

参数1：x轴占比或元素选择器

参数2：回调，可选

定点滚动

$('#demo').scrollbar('x', 0.1);

$('#demo').scrollbar('x', '.target');
"y"

参数1：y轴占比或元素选择器

参数2：回调，可选

定点滚动

$('#demo').scrollbar('y', 0.1);

$('#demo').scrollbar('y', '.target');
"top"

参数1：回调，可选

 y轴，定点滚动到顶部
"right"

参数1：回调，可选

 x轴，定点滚动到最右边
"bottom"

参数1：回调，可选

 y轴，定点滚动到底部
"left"

参数1：回调，可选

 x轴，定点滚动到最左边
"render"

参数1：参数对象，options

 再次渲染滚动条区域容器，通常应用于可变大小的容器
