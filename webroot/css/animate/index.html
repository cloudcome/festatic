2、使用说明

1. 给你需要动画效果的元素（display不能为none）上添加animated样式

2. 默认的动画时间是1S，可以改变时间和延迟时间等等，如：

#yourElement {

    -vendor-animation-duration: 3s;

    -vendor-animation-delay: 2s;

    -vendor-animation-iteration-count: infinite;

}

wendor为webkit, moz, ms, o。

3. 在需要执行动画的时候添加class即可，如fadeIn

4. 动画样式库参考了：https://github.com/daneden/animate.css
